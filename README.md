CUIT AutoCheck in<br> 
==================
每日健康打卡
默认24小时自动打卡一次
（请勿在凌晨12点到次日早上6点运行）
默认全部否。

依赖<br>
----------------
###### pip3 install requests

使用方法<br> 
-------------
###### 设置main中的账号密码地区<br> 
###### Linux 挂后台<br> 
    nohup python3 -u Autocheckin.py > chekin.log 2>&1 &
